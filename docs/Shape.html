fillRule(rule) → {Shape}

Determines how shapes are filled. If set to 'nonzero', all space within a shape will be filled. If set to 'evenodd', only areas filled an even amount of times are shown filled. See https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/fill-rule for more details.
Parameters:
Name Type Description
rule string Either 'nonzero' or 'evenodd'.
